# Planzio
Planzio adalah aplikasi perencanaan acara yang memudahkan pengguna menemukan dan memesan vendor secara cepat. Vendor dapat menampilkan layanan, harga, dan portofolio, serta memperoleh promosi melalui fitur premium. Planzio membantu vendor menjangkau lebih banyak klien dengan cara praktis dan efisien.
